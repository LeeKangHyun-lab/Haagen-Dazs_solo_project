# 📂 프로젝트명
Haagen-Dazs_solo_project


---

## 📌 프로젝트 소개
하겐다즈 웹사이트를 참고하여 만든 간단한 쇼핑몰 웹사이트 입니다.

---

## 💻 개발 환경
- **IDE** : IntelliJ IDEA / VSCode
- **언어** : Java 17
- **프레임워크** : Spring Boot 3.x
- **빌드 툴** : Gradle
- **DB** : MySQL
- **형상 관리** : GitHub

---

## 🛠 기술 스택
| 구분       | 기술 |
|------------|------|
| Backend    | Spring Boot |
| Database   | MySQL, MyBatis
| Frontend   | Thymeleaf |
| Tools      | GitHub |

---

## ⭐ 주요 기능
- 회원가입 및 로그인
- 상품 CRUD
- 파일 업로드 (이미지 첨부)
- 장바구니
- 페이지 처리
